# Resume Screening & Candidate Ranking System

## Objective

Build an NLP-based system to screen, score, and rank resumes based on a job description.

## Tools Used

- Python
- Pandas
- NLTK
- Scikit-learn
- Matplotlib

## Features

- Resume text cleaning
- Skill extraction
- Job description matching
- Similarity scoring
- Candidate ranking
- Skill gap identification

## Business Value

This project helps recruiters automate resume screening, identify suitable candidates, and speed up hiring decisions.
